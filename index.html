About Me

SOFTWARE ENGINEER

Hello! My name is Jeremiah Lu and I enjoy being hands on and turning an idea into something tangible from scratch. I've built things in various facets including home improvement, woodworking, stock market portfolios, cooking and of course, software development.

Throughout my life, I've often thought about how technology could improve various aspects of everyday life. From automation to mobile applications to machine learning, I believe that technology is constantly advancing and has the potential to change the world for the better. I am eager to be a part of this progress, particularly in the areas of automation, mobile development, and machine learning. I am excited to gain the skills and experience necessary to create innovative solutions that have a lasting impact.

Projects
Horizon Health
Horizon Health is a full-stack e-commerce application inspired by making health and self-care more accessible in one place. Users are able to browse through OTC healthcare items, find healthcare facilities nearby and participate in a live chat. React / Redux and CSS modules were used to create the UI. Flask-SQLAlchemy were used to construct the backend.
Splitwise
A clone of the popular finance tracking application, Splitwise. Users are able to track and split expenses made between them and other users. Built with React / Redux, HTML and CSS in the frontend. Flask, SQLAlchemy were used to create the backend.
Soundcloud
My first project, a clone of the popular music sharing platform, SoundCloud. Built on React / Redux, HTML, CSS, and Express. Users are able to create and edit songs, playlists and albums to their liking.
